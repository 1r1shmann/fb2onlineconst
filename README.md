# FB2 Online Constructor

Небольшой онлайн-конструктор книг в формате **FB2**.

Сделал его для случаев, когда нужно быстро собрать электронную книгу без установки редакторов, конвертеров и прочего добра. Заполнил информацию о книге, добавил главы, нажал кнопку — получил готовый `.fb2`.

## Что умеет

- указать название книги и автора;
- выбрать возрастной рейтинг;
- выбрать один или несколько жанров;
- указать язык произведения;
- добавить обложку;
- написать аннотацию;
- создать нужное количество глав;
- заполнить содержимое каждой главы;
- сгенерировать и скачать готовую книгу в формате FB2.

## Как пользоваться

1. Заполняем информацию о книге.
2. Указываем количество глав.
3. Заполняем названия и содержимое глав.
4. Нажимаем кнопку создания.
5. Скачиваем готовый FB2-файл.

Ничего сложнее этого здесь нет 🙂

## Хранение файлов

После генерации книга сохраняется на сервере и остаётся доступной для скачивания в течение суток.

Через 24 часа файлы автоматически удаляются cron-задачей.

## Технологии

Проект написан без тяжёлых фреймворков и баз данных.

Используется:

- PHP
- JavaScript
- HTML
- CSS

## Особенности

- не требует базы данных;
- работает на обычном PHP-хостинге;
- все данные хранятся в файлах;
- минимальное количество зависимостей;
- можно развернуть буквально за пару минут.

## Установка

```bash
git clone https://github.com/1r1shmann/fb2onlineconst.git
```

После этого достаточно разместить проект на сервере с поддержкой PHP.

## Зачем вообще это было сделано?

Потому что иногда хочется быстро собрать FB2-книгу из готового текста и не возиться с десктопными программами. А ещё потому что написать свой велосипед иногда быстрее и интереснее, чем искать чужой 🙂
