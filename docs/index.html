FB2CONST Online

Веб-приложение для генерации книг в формате FB2 (FictionBook 2.0).

Что умеет проект

  • Генерация FB2 через браузер
  • Поддержка глав и аннотаций
  • Жанры, язык, возрастная категория
  • Загрузка обложки (base64)
  • Скачивание готового файла
