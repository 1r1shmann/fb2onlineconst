Безопасность

Реализовано

  • htmlspecialchars() против XSS
  • проверка обязательных полей
  • ограничение размера файла (2MB)

Проблемы

  • нет CSRF защиты
  • base64 загрузка обложки
  • нет MIME проверки файлов
  • открытый доступ к /files

Рекомендации

  • finfo_file() для проверки MIME
  • CSRF tokens
  • rate limiting
  • очистка /files по cron
