Использование

  1. Откройте главную страницу
  2. Введите автора и название книги
  3. Выберите жанры, язык и категорию
  4. Добавьте аннотацию
  5. Создайте главы
  6. Нажмите «Построить FB2»

После генерации появится ссылка для скачивания файла.
